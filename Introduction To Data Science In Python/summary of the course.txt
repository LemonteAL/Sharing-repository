Course can be found at: https://www.coursera.org/learn/python-data-analysis

Summary of material covered:
"This course will introduce the learner to the basics of the python programming environment, including fundamental python programming techniques such as lambdas, reading and manipulating csv files, and the numpy library. The course will introduce data manipulation and cleaning techniques using the popular python pandas data science library and introduce the abstraction of the Series and DataFrame as the central data structures for data analysis, along with tutorials on how to use functions such as groupby, merge, and pivot tables effectively. By the end of this course, students will be able to take tabular data, clean it, manipulate it, and run basic inferential statistical analyses."


Assignment #1 Key Ideas: Basics of regular expressions in Python.

Assignment #2 Key Ideas: Basic manipulation of pandas dataframes generated from .csv files.

Assignment #3 Key Ideas: More sophisticated manipulations of pandas dataframes.

Assignment #4 Key Ideas: Additional practice using more sophisticated dataframe manipulations.
